### 记录在vscode中配置C++环境

设备说明：Mac，m系列芯片。

1. 确保环境输入`gcc -v`查看环境。
  > 出现类似输出即可 `Apple clang version 15.0.0 (clang-1500.0.40.1)`

2. 下载extensions  

    - C/C++
    - Code Runner

3. 打开设置搜索`Code-runner: Run In Terminal`。这个目的就是可以在命令行中输入数据。
